# RN navigation drawer 소개

## RN navigation drawer란?

`react-navigation-drawer`는 앱의 메인 메뉴를 보여주는 **UI panel**입니다.
유저가 앱 바에 있는 drawer icon `=`을 터치하거나 유저가  손으로 핸드폰 스크린의 왼쪽에서부터 swipe를 하면 drawer 창이 아래의 그림처럼 나타날 것입니다.
